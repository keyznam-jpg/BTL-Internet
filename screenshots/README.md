# Hướng dẫn thêm ảnh chụp màn hình

Để thêm ảnh chụp màn hình cho README.md, hãy làm theo các bước sau:

## 1. Chụp ảnh màn hình
- Chạy ứng dụng web tại `http://127.0.0.1:5000`
- Sử dụng công cụ chụp màn hình (Windows + Shift + S, hoặc Snipping Tool)
- Chụp các trang chính của hệ thống

## 2. Lưu ảnh vào thư mục này
Lưu các file ảnh với tên tương ứng:

- `login.png` - Trang đăng nhập
- `dashboard.png` - Dashboard tổng quan
- `room_map.png` - Sơ đồ phòng
- `booking.png` - Đặt phòng offline
- `online_booking.png` - Đặt phòng online
- `chat.png` - Chat real-time
- `employees.png` - Quản lý nhân viên
- `statistics.png` - Thống kê doanh thu
- `email_config.png` - Cấu hình email
- `invoice.png` - Hóa đơn thanh toán

## 3. Yêu cầu về ảnh
- **Định dạng**: PNG hoặc JPG
- **Kích thước**: Khuyến nghị 1200x800 pixels trở lên
- **Chất lượng**: Full HD (1920x1080) hoặc cao hơn
- **Nội dung**: Hiển thị đầy đủ giao diện, không crop quan trọng

## 4. Ví dụ cách chụp
- **Login**: Hiển thị form đăng nhập với logo
- **Dashboard**: Hiển thị các thống kê và menu
- **Room Map**: Hiển thị lưới phòng với màu sắc
- **Booking**: Form đặt phòng với thông tin khách
- **Chat**: Cửa sổ chat với tin nhắn mẫu

## 5. Upload lên GitHub
Sau khi thêm ảnh vào thư mục này, commit và push lên repository để ảnh hiển thị trong README.md
